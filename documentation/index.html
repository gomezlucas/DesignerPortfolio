HTML Structure

The template is based on Bootstrap Framework - http://getbootstrap.com/
The default Bootstrap grid system utilizes 12 columns, making for a 1170px wide depending on your viewport.
Below 480px viewports, the columns become fluid and stack vertically.
 
Create a .row and add the appropriate number of .col-* columns.
  
<div class="row">
        
       { Level 1 column }

        <div class="row">
            <div class="col-6"> { Level 2 } </div>
            <div class="col-6"> { Level 2 } </div>
        </div>

</div>

If you need more information, please visit this site: http://getbootstrap.com/css/#grid

JavaScript

jQuery - is a Javascript library that greatly reduces the amount of code that you must write.
For more information, please visit http://www.jquery.com/
 
All the JavaScript libraries and files are included at the bottom of every HTML page like this: 
<!-- Main Jquery JS -->
<script src="./assets/js/jquery/jquery-3.5.1.min.js" type="text/javascript"></script>        
<!-- Bootstrap JS -->
<script src="./assets/js/bootstrap/bootstrap.min.js" type="text/javascript"></script>       
<!-- Popper JS -->
<script src="./assets/js/popper/popper.min.js" type="text/javascript"></script> 
<!-- Swiper JS -->
<script src="/assets/js/swiper/swiper-bundle.min.js" type="text/javascript"></script> 
<!-- Swiper Custome JS -->
<script src="./assets/js/swiper/swiperjs-custom.js" type="text/javascript"></script> 
<!-- Aos JS -->
<script src="./assets/js/aos/aos.js" type="text/javascript"></script> 
<!-- Custom JS -->
<script src="./assets/js/custom.js"></script>

Animation on Scroll

This template uses Aos to animate the elements. To change the time of animation, settings, or disable it /assets/js/custom.js 

        //initiliaze AOS
        AOS.init(
            {
                offset: 100,
                disable: function () {
                    var maxWidth = 650;
                    return window.innerWidth > maxWidth;
                }
            }
        );

You can also can change the configuration on each element. Example: 

        <div data-aos=" fade-up" > </div> >

Find more info in the official documentation https://michalsnik.github.io/aos/

Sliders

SwipeJs slider is the slider you will find in the content. It's an essay to use lighweight carousel with some decent features.
 
Estate Section slider has class 'swiper-slide' and each slide has '<div class="portfolio__card">' wrapper for easy find where slide html code start. If you want to remove or add some slide just remove/add div with all content like this '<div class="portfolio__card">'
  
          <!-- Slider main container -->
          <div class="swiper-container">
            <!-- Additional required wrapper -->
            <div class="swiper-wrapper">
              <!-- Slides -->
              <div class="swiper-slide">
                <div class="portfolio__card">
                  <img src="./assets/images/portfolio/1.png" alt="">
                  <button class="btn btn-portfolio text-uppercase"> Desserts </button>
                </div>
              </div>
              <div class="swiper-slide">
                <div class="portfolio__card">
                  <img src="./assets/images/portfolio/1.png" alt="">
                  <button class="btn btn-portfolio text-uppercase"> cupcakes </button>
                </div>
              </div>
              <div class="swiper-slide">
                <div class="portfolio__card">
                  <img src="./assets/images/portfolio/1.png" alt="">
                  <button class="btn btn-portfolio text-uppercase"> Cakes </button>
                </div>
              </div>
            </div>
            <!-- If we need pagination -->
            <div class="swiper-pagination"></div>
      
            <!-- If we need navigation buttons -->
            <div class="swiper-button-prev"></div>
            <div class="swiper-button-next"></div>
 
If you want change option for sliders you need to open assets/js/swiper/swiperjs-custom.js and find '// Our Sliders Start' function and set true/false for option what you wont to change.
 
  
          // Swiper JS
          var mySwiper = new Swiper('.swiper-container', {
             // Optional parameters
             direction: 'horizontal',
             loop: true,
             // Navigation arrows
             navigation: {
                 nextEl: '.swiper-button-next',
                 prevEl: '.swiper-button-prev',
             },
             breakpoints: {
                 640: {
                   slidesPerView: 1,
                   spaceBetween: 20,
                 },
                 768: {
                   slidesPerView: 2,
                   spaceBetween: 40,
                 },
                 1024: {
                   slidesPerView: 3,
                   spaceBetween: 50,
                 },
               }
         
          
         })

You can find more documentation here https://swiperjs.com/demos/
